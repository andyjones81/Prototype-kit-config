What is this prototype?

This prototype example demonstartes a basic route config managed through a JSON file. It doesn't replace writing routes but makes changing them a bit easier.

It's very rudementary and can be expanded easily to do other things, like versioning the journey.

There are just a few pages with a config file driving the logic. See the file named app-journey-config.json in the data folder.
