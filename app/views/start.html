Whats going on?

The route.js file references the app-journey-config.json file and passes the appropropriate object (appJourney) to the view.

Back link references the config and uses the back value to set the url.

    
<a href="{%raw%}{{appJourney.back}}{%endraw%}" class="govuk-back-link">Back</a>

Form action references the config and uses the submit value to set the url (you might want a different submit url to the page).

        
<form method="POST" action="{%raw%}{{appJourney.submit}}{%endraw%}">
